# ALDS1

<https://judge.u-aizu.ac.jp/onlinejudge/finder.jsp?course=ALDS1>

## スキップする問題

- [1_2_D Shell Sort](https://judge.u-aizu.ac.jp/onlinejudge/description.jsp?id=ALDS1_2_D&lang=ja): 途中過程を出力するのがめんどくさいのでスキップ。shell sort自体は別リポジトリで実装した。
- [1_4_B Binary Search](https://judge.u-aizu.ac.jp/onlinejudge/description.jsp?id=ALDS1_4_B&lang=ja): 1_4_Aで実装済みなのでスキップ。
- [1_5_B Merge Sort](https://judge.u-aizu.ac.jp/onlinejudge/description.jsp?id=ALDS1_5_B&lang=ja): ちゃんとソートできているが、交換した回数が合わないのでスキップ。
- [1_6_A counting sort](https://judge.u-aizu.ac.jp/onlinejudge/description.jsp?id=ALDS1_6_A&lang=ja): メモリ制限オーバーになるのでスキップ。
  - Listを使ってみた
  - arrayにしてみた
  - Vector + ByteStringにしてみた
- [1_6_C quick sort](https://judge.u-aizu.ac.jp/onlinejudge/description.jsp?id=ALDS1_6_C&lang=ja): quick sort自体は別リポジトリで実装したのでスキップ。
- [1_6_B Partition](https://judge.u-aizu.ac.jp/onlinejudge/description.jsp?id=ALDS1_6_B&lang=ja): `filter`で実装したが、in-placeでないのでスキップ。
